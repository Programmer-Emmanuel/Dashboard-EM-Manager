À propos du dashboard

EM-MANAGER Dashboard Administrateur

Interface d'administration complète pour gérer les administrateurs et les entreprises depuis un tableau de bord centralisé.

Ce que vous pourrez faire

Le dashboard admin donnera accès à toutes ces fonctionnalités

Gestion des administrateurs

  • Ajouter de nouveaux administrateurs
  • Voir la liste complète des admins
  • Consulter les détails d'un admin
  • Supprimer des administrateurs

Gestion des entreprises

  • Lister toutes les entreprises
  • Voir les détails d'une entreprise
  • Consulter les soldes des entreprises
  • Supprimer des entreprises

Gestion du profil

  • Voir ses informations personnelles
  • Modifier son profil
  • Changer son mot de passe

Sécurité et contrôles

  • Authentification sécurisée
  • Réinitialisation des mots de passe
  • Session avec tokens Sanctum

En résumé

Le dashboard admin permettra de gérer les administrateurs (ajout, modification, suppression), gérer les entreprises (liste, détails, soldes, suppression), et gérer son propre profil (informations, mot de passe) - le tout de manière sécurisée avec authentification.
